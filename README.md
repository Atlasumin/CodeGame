# Code Game

Code GAME 是一个通过编写 AI 脚本控制坦克进行比赛的游戏。

http://codegame.org

该版本库为 codegame.org 平台的全部代码。

娱乐第一，请点右上角 star！

## 安装

    $ npm install
    $ npm run build
    $ cp config/_sample.json config/development.json
    $ vim config/development.json
    $ node app

## 技术栈

Node.js, MySQL
